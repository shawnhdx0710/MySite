I built an interactive shot chart to explore player shooting patterns. The chart renders on a `<canvas>` element inside a React component — and because this site uses MDX, I can embed it directly in this blog post.

## How it works

Each dot represents a shot attempt. Green dots are makes, red are misses. The data is synthetic for now, but swapping in real NBA shot-location data from the NBA stats API is straightforward.
